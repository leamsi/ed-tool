# ed-tool

This is a quick an easy tool to edit files that doesn't require context matching
so it works more often than not (compared to your `edit` tool).

## venv

Use a virtual environment (`.venv`) to install any dependencies (like `pytest`
for testing) and to run the tests.

## README.md

Read the README.md for instructions on how to use the tool so you get a clear
idea of what is supposed to do and how.
